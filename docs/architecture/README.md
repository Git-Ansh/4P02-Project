# System Architecture

This directory contains architecture documentation for the CodeIntegrity system.

## Overview

The system follows a three-tier architecture:

1. **Presentation Layer** - React.js frontend
2. **Application Layer** - FastAPI backend
3. **Data Layer** - PostgreSQL database

## Components

- [Analysis Engine](./analysis-engine.md) - Code similarity detection
- [Submission Handler](./submission-handler.md) - File upload and processing
- [Repository Manager](./repository-manager.md) - Code repository management
